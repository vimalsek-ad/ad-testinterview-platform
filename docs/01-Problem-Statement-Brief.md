# Problem Statement Brief — ADHire
*Alter Domus in-house candidate assessment platform (replacing 4–5 external tools)*

---

### Customer
**Alter Domus** — global fund & corporate services firm (~5,000+ employees, **40+ offices** across EU, US, APAC, LATAM). Hiring is run by **15+ hiring teams** — Engineering, Data, DevOps, Finance, Compliance, Operations — each recruiting independently at global scale.

### Problem
Alter Domus assesses candidates using **4–5 disconnected external vendor tools** (HackerRank, Codility, HireVue, SHL/Criteria, plus ad-hoc). The tools are **not integrated**, which creates four compounding problems:

- **High, recurring vendor cost** — per-seat SaaS licences across five products.
- **Heavy manual hiring-ops effort** — recruiters stitch results together and evaluate candidates by hand.
- **No built-in proctoring** — limited assessment integrity and no audit trail.
- **Zero data portability** — candidate PII lives inside vendor systems, creating GDPR / data-residency / compliance risk and vendor lock-in.

The result is a **fragmented candidate experience** (4 separate tools per candidate) and hiring data the business does not own or control.

### Data (current scale)

| Dimension | Today |
|---|---|
| Assessments | 500+/month · 6,000+/year |
| Open roles | 80–120/month · 5–8 candidates assessed each |
| Hiring teams | 15+ |
| Global reach | 40+ offices (EU/US/APAC/LATAM) |
| Tooling | 4–5 vendors, none integrated |

### Business Case
Bring candidate assessment **in-house** — a single **AD-branded portal on Alter Domus infrastructure** — that replaces the vendor stack end-to-end: question bank, coding + interview assessments, **built-in proctoring**, automated + AI-assisted scoring, human-in-the-loop review, and a full audit trail. This removes SaaS licence spend, collapses manual evaluation time, and keeps **100% of candidate data on AD infrastructure** (in-region, GDPR-aligned). Projected **€475K+ annual value, ~15× return, break-even by Month 3.**

### Definition of success
ADHire replaces the external stack such that evaluation time drops to **~15 min/candidate**, marginal licence cost is **€0** (unlimited assessments), proctoring + an **immutable audit trail** are built in, and **100% of candidate data stays on AD infrastructure** — with break-even inside the first quarter.

*Figures are planning estimates scaled to Alter Domus (~5,000+ employees, 40+ offices). Loaded people-cost assumed at €50/hr.*
